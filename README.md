# 🚀 Automated Financial Due Diligence Pipeline: Apple Inc. (AAPL)

Dieses Repository enthält eine vollautomatisierte **Data & Execution Pipeline** zur Extraktion, Berechnung, Visualisierung und Auswertung einer hochprofessionellen Bilanzanalyse (5-Jahres-Trend) für Apple Inc. 

Anstatt Daten manuell in Excel zu berechnen, orchestriert dieses Projekt den gesamten Workflow eines Senior Financial Analysts – von der rohen Datenbank bis zum druckfertigen Analysten-Report – mit einem einzigen Befehl.

## ⚙️ Quickstart: Die 1-Klick-Pipeline

Um die komplette Analyse auszuführen, alle Daten neu zu berechnen und den finalen Report zu generieren, starte einfach den Orchestrator im Terminal:

```bash
python pipeline.py
